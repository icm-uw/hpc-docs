## vim

* `:w` - 'write'
* `:wq!` - 'write and quit'
* `:q!` - 'quit without saving (good for mistakes)
* `i` - insert! (Insert mode, Replace mode)
* `Esc` - back to command mode
* `u` to undo in command mode
